### Basic Setup

### Customization

#### Change Colors
Edit the CSS variables in the `:root` selector:
```css
:root {
    --primary: #6d4c41;        /* Your brand color */
    --secondary: #8d6e63;      /* Secondary color */
    --accent: #a1887f;         /* Accent color */
}
```

#### Modify Parallax Speed
Adjust the multiplier in the JavaScript:
```javascript
const speed = (index + 1) * 0.5;  // Change 0.5 to your preferred value
```

```css

    50% { transform: translateY(-20px) scale(1.02); }
}
```
